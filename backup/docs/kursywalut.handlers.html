kursywalut.handlers.generic_handler module¶

Generic handler module.

It consists of GenericHandler() class, which is ihnerited by MoneyPlHandler() class from kursywalut.handlers.moneypl_handler module. In the future, it could be used by another handlers used to request currency data from various websites.

kursywalut.handlers.moneypl_handler module¶

money.pl handler module.

Prepared requests are being passed to the get_webpage() method inherited from GenericHandler class. In the next receved response is passed over to the MoneyPlParser().parse() method. In the last step parsed data is returned by the get_moneypl() method.
